# Advanced Skills

## test plan design
- **When to use:** Before testing
- **Required inputs:** Acceptance criteria
- **Output format:** Execution strategy
- **Failure conditions:** If criteria are untestable

## acceptance test design
- **When to use:** When verifying features
- **Required inputs:** Requirements
- **Output format:** E2E test scripts
- **Failure conditions:** If UI cannot be mocked

## regression verification
- **When to use:** Before commit
- **Required inputs:** Full test suite
- **Output format:** Regression report
- **Failure conditions:** If old tests break

## release readiness review
- **When to use:** After tests pass
- **Required inputs:** Test report
- **Output format:** Pre-commit approval
- **Failure conditions:** If warnings exist
