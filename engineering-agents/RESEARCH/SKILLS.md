# Advanced Skills

## deep codebase tracing
- **When to use:** When understanding a complex bug
- **Required inputs:** Error stack trace
- **Output format:** Root cause analysis report
- **Failure conditions:** If trace leaves the repository

## external docs review
- **When to use:** When evaluating a new dependency
- **Required inputs:** Package name
- **Output format:** Dependency risk report
- **Failure conditions:** If documentation is unavailable
