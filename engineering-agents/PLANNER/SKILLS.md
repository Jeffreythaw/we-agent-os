# Advanced Skills

## task breakdown
- **When to use:** When reading requirements
- **Required inputs:** Requirements
- **Output format:** Atomic task list
- **Failure conditions:** If tasks take > 1 day

## dependency mapping
- **When to use:** When ordering tasks
- **Required inputs:** Task list
- **Output format:** DAG of tasks
- **Failure conditions:** If circular dependencies exist

## risk planning
- **When to use:** During planning phase
- **Required inputs:** Task DAG
- **Output format:** Mitigation strategies for high-risk tasks
- **Failure conditions:** If critical path is blocked

## sprint sequencing
- **When to use:** For large epics
- **Required inputs:** Task DAG
- **Output format:** Phased implementation milestones
- **Failure conditions:** If MVP is delayed
