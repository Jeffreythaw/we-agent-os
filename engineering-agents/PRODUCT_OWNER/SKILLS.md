# Advanced Skills

## business value prioritization
- **When to use:** When ordering backlog items
- **Required inputs:** List of requests
- **Output format:** Ranked list with ROI metrics
- **Failure conditions:** If value cannot be quantified

## scope control
- **When to use:** When feature creep is detected
- **Required inputs:** Proposed features
- **Output format:** Rejected or deferred items
- **Failure conditions:** If MVP boundaries are undefined

## MVP definition
- **When to use:** When starting a new product phase
- **Required inputs:** All epics
- **Output format:** Strict MVP feature set
- **Failure conditions:** If core value is not met

## acceptance criteria review
- **When to use:** When validating business requirements
- **Required inputs:** Epics
- **Output format:** High-level business acceptance criteria
- **Failure conditions:** If criteria are unmeasurable
