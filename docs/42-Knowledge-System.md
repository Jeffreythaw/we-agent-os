# Knowledge System

## 1. Concept
Knowledge is reusable reference material. It differs from Memory in that Knowledge is generally static, curated, and provided as "ground truth" (e.g., API documentation, company style guides, brand assets).

## 2. Implementation
Knowledge is typically stored in a Vector DB or indexed file system and injected into the Context Engine via RAG (Retrieval-Augmented Generation) when relevant Skills are invoked.
