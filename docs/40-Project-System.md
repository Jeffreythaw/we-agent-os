# Project System

## 1. Concept
A Project acts as the specific boundary for a client, job, or objective within a Workspace. While a Workspace might be "My Agency", Projects would be "Client A Website" and "Client B Marketing".

## 2. Encapsulation
Projects encapsulate:
- Personas tailored to the job.
- Knowledge specific to the client.
- Artifacts produced during execution.
- Executions representing the history of work done.
