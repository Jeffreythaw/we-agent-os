# Artifact System

## 1. Concept
Artifacts represent the tangible outputs of the OS: email drafts, markdown reports, source code files, PDFs, images. 

## 2. Distinction from Memory
- **Memory** is transient or internal state used by the agent to think.
- **Artifacts** are the *deliverables* meant for human consumption or external system use.

## 3. Versioning
Artifacts must be versioned. If a human edits an artifact, the OS must recognize the change before continuing work to prevent destructive overwrites.
