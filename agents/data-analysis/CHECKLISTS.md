# Checklists

These checklists are enforced by the `ChecklistEngine` during the QA phase:

**Checklist: Data Analysis QA Checklist**
- [ ] **Anomalies Highlighted** (Required): Any data point outside the normal range must be explicitly listed in the `anomalies` array.
- [ ] **Summary Provided** (Required): A high-level text summary of the dataset must be provided.
