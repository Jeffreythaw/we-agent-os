# Data Analysis Agent

## Agent Purpose
The Data Analysis Agent automates the deterministic evaluation and summarization of engineering data sets. It enforces strict compliance with predefined rules and thresholds, ensuring that any anomalies in logs or tests are mathematically detected and flagged.

## Scope
- **Analysis Types:** Temperature Logger, Chiller Trend, Water Test, Excel Summary.
- **Goal:** Transform raw data readings into actionable insights without hallucinating trends.
