# Development Lifecycle

## 1. Purpose
Define the mandatory sequence of phases for building any capability within WE Agent OS.

## 2. Scope
Applies to human engineers and AI agents contributing to Business Agents or Framework modules.

## 3. Rules & Guidelines
1. **Requirements**: Define the explicit goal.
2. **Planning**: Formulate a step-by-step implementation plan. Do not write code.
3. **Design**: Produce architectural schemas, ADRs, or Spec documents.
4. **Implementation**: Write deterministic code strictly within boundaries.
5. **Testing**: Run unit and regression tests.
6. **Deployment**: Release the capability.
7. **Documentation**: Update the `context/` foundation.
8. **Maintenance**: Enforce backward compatibility.
